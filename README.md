## 🚀 Sobre o projeto

Este projeto consiste no desenvolvimento de uma landing page inspirada no AluraBooks, com foco na construção de layouts responsivos e na aplicação de boas práticas de HTML e CSS.

A página foi estruturada utilizando HTML semântico e estilizada com CSS moderno, priorizando a adaptação para diferentes dispositivos através da abordagem mobile-first.
